# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Professional GitHub templates (Issues, PRs, Workflows).
- Comprehensive `setup.ps1`, `setup.sh`, and `START.bat` for one-click setup.
- `scripts/` directory for better organization of utility scripts.
- Code of Conduct, Security Policy, and Authors documentation.

### Changed
- Reorganized project structure: moved all root-level batch/shell scripts to `scripts/`.
- Updated `README.md` to reflect the new professional structure.

## [1.0.0] - 2026-01-27

### Added
- Initial release of Setu Voice-to-ONDC Gateway.
- Core Voice Injector component.
- Visual Verifier component.
- Network Simulator.
- Docker and Docker Compose setup.
- Integration with Vercel AI SDK and OpenAI.
- Local SQLite and Production PostgreSQL support.
