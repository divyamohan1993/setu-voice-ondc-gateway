Project Abstract

The Problem

India's digital commerce revolution (ONDC) is bypassing its most vital sector: agriculture. Despite 260 million farmers, the vast majority remain "invisible" to the digital economy. The barriers are structural: linguistic diversity (interfaces are primarily in English), low digital literacy (inability to navigate complex forms), and technical complexity (managing catalogs and inventory). Consequently, farmers are forced to sell to local middlemen at a fraction of fair market value, unable to access the broader national market.

The Solution

Setu bridges this divide by replacing complex graphical interfaces with the most natural human interface: Voice. It is a production-grade gateway that enables illiterate farmers to list produce on the ONDC network simply by speaking in their native language.

How It Works

  1. Voice Input: The farmer taps a single button and speaks naturally (e.g., "I have 500kg of Nashik Onions to sell at 25 rupees").
  2. AI Processing: Powered by Google Gemini 3 Flash, the system transcribes the speech, translates it from regional languages (Hindi, Marathi, Tamil, etc.) to English, and intelligently extracts structured data (Commodity, Quantity, Price, Quality).
  3. Protocol Conversion: The system maps this data to the Beckn Protocol (v1.2.0), generating valid JSON schemas required for the ONDC network.
  4. Network Broadcast: The catalog is instantly broadcast to the open network, making it visible to thousands of potential buyers (retailers, wholesalers) simultaneously.

Key Innovations

  • Multilingual AI Core: Leveraging Gemini's advanced context window to handle dialect nuances and mixed-language speech (Hinglish/Tanglish).
  • Zero-UI Design: Removing the need for typing or reading, making digital commerce accessible to the completely illiterate.
  • Real-Time Mandi Integration: Fetches live government price data (Agmarknet) to suggest fair pricing to farmers before they list.

Impact

Setu democratizes access to the digital economy. It transforms the mobile phone from a consumption device into a powerful economic tool, ensuring that the benefits of ONDC reach the "last mile"â€”the Indian farmer.

Software Stack

Frontend Framework & UI

  • Next.js 15 (App Router): Core framework for server-side rendering and API routes.
  • React 19: library for building the user interface.
  • Tailwind CSS 4: Utility-first CSS framework for styling.
  • Framer Motion: Library for fluid animations and micro-interactions.
  • Lucide React: Icon set for consistency.
  • Radix UI: Headless UI primitives for accessible components.

Backend & Database

  • Node.js: Runtime environment.
  • Prisma ORM: Type-safe database client.
  • PostgreSQL: Production database (relational data).
  • SQLite: Development database.
  • Zod: Schema validation for API requests.

AI & Machine Learning

  • Google Gemini 3 Flash: Large Language Model (LLM) for voice-to-text translation and entity extraction.
  • Vercel AI SDK: Middleware for streaming AI responses.

Protocols & Infrastructure

  • Beckn Protocol (v1.2.0): Open standard used for ONDC compliance.
  • Docker & Docker Compose: Containerization for consistent deployment.
  • Google Cloud Platform (GCP): Hosting and compute infrastructure.
  • Vitest: Unit and integration testing framework.
